# Описание содержимого дириктории

## RAW

$Сырые$ данные, устанавливыемые с Kaggle (`https://www.kaggle.com/datasets/nikhil1e9/loan-default?resource=download`)<br>
`RAW/Loan_default.csv` файл, устанавливающийся в результате запуска скрипта `src/data/download.py`. Содержит данные исходного датасета<br>
`RAW/Loan_default_example.csv` файл, содержащий примеры записей `RAW/Loan_default.csv`<br>

## PROCESSED

$Обработанные$ данные, (данные полученные в результате обработки исходного датасета скриптом `src/data/process.py`)<br>
`PROCESSED/LoanDefaultPredictionDatasetPROC.csv` файл, содержащий обработанные данные датасета `RAW/Loan_default.csv` указаным выше скриптом. Содержит данные исходного датасета<br>
`PROCESSED/LoanDefaultPredictionDatasetPROC_example.csv` файл, содержащий примеры записей `PROCESSED/LoanDefaultPredictionDatasetPROC.csv`<br>
